# 🛡 VoiceShield AI

Intelligent scam detection system for Telegram based on Machine Learning.

## 🚀 Features

- Text scam detection (TF-IDF + ML)
- Voice message transcription (Whisper)
- Audio feature analysis (MFCC)
- Combined risk scoring
- Real-time Telegram bot

## 🧠 Technologies

- Python
- aiogram
- scikit-learn
- librosa
- Whisper
- Telegram Bot API
