# 🛡 VoiceShield AI

**VoiceShield AI** — интеллектуальная система обнаружения мошеннических сообщений и звонков в Telegram с использованием методов машинного обучения.

Проект анализирует **текстовые и голосовые сообщения**, определяя вероятность того, что сообщение является мошенническим (скам).
Система использует **мультимодальный подход**, объединяя анализ текста, аудио и распознавание речи.

---

# 🚀 Возможности

* 📩 Анализ текстовых сообщений
* 🎙 Анализ голосовых сообщений
* 🧠 Распознавание речи (Speech-to-Text)
* 🔊 Анализ аудио-фич (MFCC)
* 🤖 Модель машинного обучения для определения скама
* ⚡ Telegram-бот для работы в реальном времени
* 📊 Вероятностная оценка риска мошенничества

---

# 🧠 Как работает система

VoiceShield AI использует **несколько этапов анализа**:

### 1️⃣ Распознавание речи

Голосовое сообщение преобразуется в текст с помощью модели **Whisper**.

### 2️⃣ Анализ текста

Текст обрабатывается с помощью модели машинного обучения.

Модель оценивает вероятность того, что текст содержит признаки мошенничества.

### 3️⃣ Анализ аудио

Из аудио извлекаются **MFCC-признаки**, которые используются для определения возможных признаков синтетической или мошеннической речи.

### 4️⃣ Финальная оценка

Результаты текстового и аудиоанализа объединяются для вычисления **итоговой вероятности мошенничества**.

# ⚙ Установка

### 1️⃣ Клонировать репозиторий

```
git clone https://github.com/yourusername/VoiceShieldAI.git
cd VoiceShieldAI
```

---

### 2️⃣ Создать виртуальное окружение

```
python -m venv venv
```

Активировать:

**Windows**

```
venv\Scripts\activate
```

---

### 3️⃣ Установить зависимости

```
pip install -r requirements.txt
```

---

# 🔑 Настройка Telegram-бота

Создайте файл `.env` в корне проекта:

```
BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
```

Токен можно получить через **BotFather** в Telegram.

---

# ▶ Запуск бота

Перейдите в папку проекта и выполните:

```
python -m src.bot.telegram_bot
```

Если всё работает, вы увидите:

```
Loading models...
Models loaded successfully.
Bot started...
```

Теперь бот готов принимать сообщения.

# 🧪 Обучение моделей

### Обучение текстовой модели

### Обучение аудио модели
